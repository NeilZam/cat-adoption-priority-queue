# Cat Adoption Priority Queue System

A heap-based priority queue application written in C for managing animal adoption and triage workflows.

## Features

- Min-Heap implementation
- Max-Heap implementation
- Priority-based animal selection
- Heap insertion and removal
- Dynamic priority updates
- Efficient O(log n) operations

## Concepts Used

- Heaps
- Priority Queues
- Dynamic Memory Allocation
- Algorithm Optimization

## Technologies

- C
- GCC
- Command Line

## What I Learned

This project helped me understand heap operations, priority queue design, and efficient data management techniques.
